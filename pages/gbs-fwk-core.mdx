## Core

Core provides various utility services for `building block`.

### Usage

Import `gbs-fwk-core` to use it. There is no need for additional installtion of `core` since it will already shipped with `gbs-fwk-buildingblock`.

```bash copy
import {service_name} from 'gbs-fwk-core'
```
